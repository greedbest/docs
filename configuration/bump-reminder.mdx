## Overview

greed's bump reminder helps your server stay at the top of DISBOARD by sending a reminder message exactly 2 hours after a successful `/bump`. It also includes a "thank you" system and a leaderboard to encourage activity.

## Getting Started

Enable the bump reminder system in your server.

<CodeGroup>
```javascript Syntax
,bumpreminder enable
```
</CodeGroup>

<Info>
  greed automatically detects successful bumps from DISBOARD. You don't need to configure anything else to start tracking.
</Info>

## Customization

### Reminder Message
Customize the message greed sends when it's time to bump.
```javascript
,bumpreminder reminder It's time to bump the server! Type `/bump` to help us grow.
```

### Thank You Message
Customize the message greed sends immediately after someone successfully bumps.
```javascript
,bumpreminder thankyou Thanks for bumping, {user}! You now have {bumps} total bumps.
```

## Management & Stats

### Viewing Status
Check the current settings and time remaining until the next bump.
```javascript
,bumpreminder view
```

### Leaderboard
See who the top bumpers are in your community.
```javascript
,bumpreminder leaderboard
```

### Testing
Send a test reminder to see how it looks in the current channel.
```javascript
,bumpreminder test
```

<Tip>
  Use `,bumpreminder disable` to turn off the system entirely.
</Tip>
