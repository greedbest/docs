## Introduction

greed is a multipurpose Discord bot designed to keep your server safe and easy to manage. This guide will help you set up the core features and get the bot running in your community.

<Info>
  The default prefix for greed is `,`. Use `,prefix set (symbol)` to change it for your server. You can also mention the bot to see your current prefix.
</Info>

## Quick Start Guide

<Steps>
  <Step title="Invite greed to your server">
    Add the bot with administrative permissions to ensure all security and moderation tools function correctly.
  </Step>
  <Step title="Run the setup command">
    Use `,setup` to automatically create necessary roles (imute, rmute, jailed) and the greed-mod channel category.
  </Step>
  <Step title="Configure your first module">
    Start by enabling antinuke or setting up your custom invoke messages to personalize your server's interactions.
  </Step>
</Steps>
